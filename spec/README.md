# AEVA Eat — техзадание (MVP)

Камерный invite-only дневник еды для своих. Скрапбук + поиск + карта.

## Состав

### Зафиксированные решения

| Файл | Что внутри |
|---|---|
| `DESIGN-DECISIONS.md` | Закрытые решения дизайнера (verbatim, на них опирается код) |
| `STATUS.md` | Исторический журнал по раундам разработки (R1–R5.1) |

> Актуальное состояние и история фич — `CHANGELOG.md` и `ROADMAP.md` в корне репозитория.

### Изначальное ТЗ (для контекста)

| Файл | Что внутри |
|---|---|
| `product.md` | Философия, экраны, флоу |
| `design.md` | Скрапбук-система: бумага, шрифты, артефакты |
| `backend.md` | Дельта к API |
| `mvp-scope.md` | Что в MVP, что в next, что выкидываем |

## Скрапбук-дизайн

Дизайн-система реализована в `frontend/src/assets/scss/scrapbook.scss` и vue-примитивах
в `frontend/src/components/scrapbook/` (полароид, скотч, штамп, билетик, gem, kruzhok).
Деталь системы — в `design.md`.

## Что не входит в MVP

- Нативный мобильный клиент (PWA — да; iOS/Android-нативки — нет)
- Транскод видео
- Тёмная тема
- pgAdmin / админка (используем существующее)
- CI/CD

## Приоритет работ

1. Дизайн-система (CSS-переменные, шрифты Lora+Caveat, базовые компоненты — полароид/скотч/штамп/билетик/gem)
2. Главный экран `Доска` (см. `product.md` → §Экраны → Доска)
3. Бэкенд-дельта: `gems`, `cities`, `users/:id` страницы (см. `backend.md`)
4. Экраны: `Найти`, страница `Город`, страница `Профиль`, hub `Жемчужины`
5. Экран `Я` (профиль + wishlist + заметки)
6. Карта — рестайл маркеров под штампики
7. Анимации (gem-блик, страница месяца)

Отдельно — миграция вёрстки текущих экранов (`Place.vue`, `PlaceForm.vue`, `Profile.vue`) под скрапбук. План см. в `design.md` → §Миграция.
