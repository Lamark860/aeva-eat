Камерный invite-only дневник еды для своих. Скрапбук + поиск + карта.

## Прототип

В корне репозитория этого проекта — `index.html` с интерактивным макетом главного экрана (`v2`). Запускать:
открыть `index.html` в браузере, либо посмотреть на канвасе. JSX/CSS — там же, могут быть полезны как референс для разраба.

- `scrapbook.css` — все примитивы скрапбук-системы (полароид, скотч, штамп, билетик, печать, gem, kruzhok)
- `variants.jsx` — общие компоненты + ранние версии для сравнения
- `v2.jsx` — финальная версия главного экрана

## Что не входит в MVP

- Нативный мобильный клиент (PWA — да; iOS/Android-нативки — нет)
- Транскод видео
- Тёмная тема
- pgAdmin / админка (используем существующее)
- CI/CD

## Приоритет работ

1. Дизайн-система (CSS-переменные, шрифты Lora+Caveat, базовые компоненты — полароид/скотч/штамп/билетик/gem)
2. Главный экран `Доска` (см. `product.md` → §Экраны → Доска)
3. Бэкенд-дельта: `gems`, `cities`, `users/:id` страницы (см. `backend.md`)
4. Экраны: `Найти`, страница `Город`, страница `Профиль`, hub `Жемчужины`
5. Экран `Я` (профиль + wishlist + заметки)
6. Карта — рестайл маркеров под штампики
7. Анимации (gem-блик, страница месяца)

Отдельно — миграция вёрстки текущих экранов (`Place.vue`, `PlaceForm.vue`, `Profile.vue`) под скрапбук. План см. в `design.md` → §Миграция.
