# AEVA Eat — дизайн-система (скрапбук, mobile-first)

Все стили — в `scrapbook.css` (см. корень проекта). JSX-примитивы — в `variants.jsx` и `v2.jsx`.

## Палитра

```css
--paper:        oklch(0.965 0.018 82);   /* фон бумаги */
--paper-deep:   oklch(0.93 0.022 82);
--paper-edge:   oklch(0.88 0.03 82);
--ink:          oklch(0.22 0.02 60);     /* основной текст */
--ink-soft:     oklch(0.42 0.02 60);
--ink-mute:     oklch(0.6 0.015 60);
--terracotta:   oklch(0.58 0.14 30);     /* акцент 1 — гем, маркер, призывы */
--ochre:        oklch(0.66 0.13 85);     /* акцент 2 — gem-fill */
--moss:         oklch(0.55 0.09 145);
--plum:         oklch(0.5 0.10 350);
```

Скотч — полупрозрачные пастели тех же оттенков (`--tape-ochre`, `--tape-rose`, `--tape-mint`, `--tape-blue`).

## Шрифты

- **Lora** (serif) — основная, тело + заголовки. Italic 500 для заголовков-секций.
- **Caveat** (handwritten) — даты, рейтинги в билетиках, рукописные пометки, подписи-капшоны.
- Капс с разрядкой 0.18em — штампы (Lora 600 uppercase).

Кириллица в обоих — Google Fonts.

## Принципы

- **Раскладка строгая, артефакты живые.** Сетка/поток понятные. Внутри артефактов — наклоны, скотч, рукописные пометки. Ничего хаотичного на верхнем уровне.
- **Углы наклона 1-5°.** Никогда больше — иначе выглядит детски.
- **Скотч обязателен** на полароидах и записках — это «приклеено». Без скотча → плоско → колхоз.
- **Аватарка автора в углу каждого артефакта** — `AuthorTag`, кружок 22px с инициалом.
- **Жемчужины сильно подсвечены** — gem-знак с анимацией. Не мелкий значок, а сразу читаемая «звёздочка».
- **Ритм в ленте.** Никакая запись не повторяет предыдущую. Размеры меняются, типы артефактов чередуются (полароид / только записка / только билет / кружочек).
- **Анимации мягкие.** Gem-gleam — раз в ~5 сек, плавно. Никаких пружин и баунсов.

## Иконки

- Минимум. Эмодзи запрещены в продакшне (placeholder в макете).
- В тулбаре нижнем — глифы из Caveat (`✦ ◎ ⌕ ✺`) или из Tabler Icons (открытый набор) в линейном стиле, тонкие 1.5px.
- Маркеры на карте — кастом SVG под штампик (см. `screen-map.md` в продукте).

## Миграция с Bootstrap 5

**Что оставить с Bootstrap:**
- Формы (input, select, button) — пока. На MVP — перестилизовать через CSS-override (бумажный фон, серифа, тонкая граница). Потом — выпилить полностью.
- Гриды/утилиты — оставить как fallback.
- Модалки — оставить, перестилизовать рамку под бумагу.

**Что переписать с нуля:**
- `PlaceCard.vue` → `<ArtefactCard>` (полароид + автор + билетик)
- `Home.vue` → `<DoskaScreen>`
- Список отзывов в `PlaceDetail.vue` → лента развёрнутых артефактов
- Навигация (вместо bootstrap-навбара) — кастомный `wordmark` сверху + `.tabbar` снизу

**Шрифты:**
- Подключить через Google Fonts (preconnect + display=swap)
- Системный fallback: `Georgia, "Times New Roman", serif` для серифы; `cursive` для рукописного

## Ссылка на референс

Финальный макет главного — артборд `v2 · единый скрапбук` в `index.html`. Все размеры/отступы/тени можно мерить прямо в DOM.

## Что не делаем в MVP

- Сложные иллюстрации, рисованные элементы — нет, всё CSS/SVG-примитивы
- Лотти/сложные анимации — нет, только CSS keyframes
- Тёмная тема — нет (но переменные можно проектировать так, чтобы потом легко добавить)
- Кастомные шрифты помимо Google Fonts — нет
