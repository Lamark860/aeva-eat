## Запланировано

### Админка
- [ ] Панель администратора (superuser): CRUD кухонь, категорий, тегов из UI
- [ ] Управление пользователями: блокировка, смена роли, сброс пароля
- [ ] Модерация отзывов: просмотр всех, удаление неуместных
- [ ] Дашборд со статистикой: кол-во заведений/отзывов/юзеров, графики по датам

### Инфраструктура
- [ ] pgAdmin в docker-compose (веб-интерфейс для БД)
- [ ] Rate limiting на API (защита от спама)
- [ ] Транскодирование видео через ffmpeg (серверная оптимизация размера)

### Заведения
- [ ] Статус заведения: открыто / временно закрыто / закрылось навсегда
- [ ] Несколько фото к заведению (галерея, не одно image_url)
- [ ] Часы работы (расписание по дням)
- [ ] Средний чек (диапазон цен)
- [ ] Теги от пользователей (например: «есть веранда», «подходит для свиданий», «шумно»)
- [ ] Ссылка на меню / фото меню
- [ ] Контакты (телефон, Instagram, Telegram)

### Отзывы
- [x] Редактирование / удаление отзыва из профиля
- [ ] Фото-галерея к отзыву (несколько фото, не одно)
- [ ] Комментарии к отзывам (обсуждение с друзьями)

### Пользователи
- [x] Смена пароля
- [x] Загрузка аватарки
- [ ] Публичные профили (посмотреть отзывы другого пользователя)
- [ ] Персональная статистика: любимые кухни, средние оценки, топ заведений
- [ ] Лента активности друзей (кто куда сходил)

### UI/UX
- [x] Пагинация списка заведений (сейчас грузятся все разом)
- [ ] Тёмная тема
- [ ] PWA (иконка на телефоне, оффлайн-просмотр)
- [ ] Мобильная адаптация (bottom nav, свайпы)

### Деплой
- [ ] CI/CD (GitHub Actions: линт + тесты + деплой)
- [ ] Продакшн-деплой на VPS
- [ ] Бэкап БД по расписанию (cron + pg_dump в S3/local)
