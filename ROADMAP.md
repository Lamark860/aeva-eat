## Запланировано

### Инфраструктура
- [ ] pgAdmin в docker-compose (веб-интерфейс для БД)
- [ ] RBAC — роли и права доступа (admin / editor / viewer)
- [ ] Rate limiting на API (защита от спама)

### Заведения
- [ ] Статус заведения: открыто / временно закрыто / закрылось навсегда
- [ ] Несколько фото к заведению (галерея, не одно image_url)
- [ ] Часы работы (расписание по дням)
- [ ] Средний чек (диапазон цен)
- [ ] Теги от пользователей (например: «есть веранда», «подходит для свиданий», «шумно»)

### Отзывы
- [ ] Редактирование / удаление отзыва из профиля
- [ ] Фото-галерея к отзыву (несколько фото, не одно)

### Пользователи
- [ ] Смена пароля
- [ ] Загрузка аватарки
- [ ] Публичные профили (посмотреть отзывы другого пользователя)

### UI/UX
- [ ] Пагинация списка заведений (сейчас грузятся все разом)
- [ ] Тёмная тема
- [ ] PWA (иконка на телефоне, оффлайн-просмотр)

### Деплой
- [ ] CI/CD (GitHub Actions: линт + тесты + деплой)
- [ ] Продакшн-деплой на VPS
