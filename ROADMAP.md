## Ждут решения дизайнера

См. `spec/OPEN-QUESTIONS.md` — все живут на pragmatic-дефолтах, можно использовать как есть.

- **R5-Q1** — гендер глагола в подписи жемчужины (эвристика `-а/-я → fem`)
- **R5-Q2** — место без отзывов на Доске сейчас не отображается вообще
- **R5-Q3** — `/p/:id` после логина → редирект на Доску, а не на конкретное место
- **R5-Q4** — wishlist-карточки на Доске без tilt
- **R5-Q5** — постер видео-кружка на iOS Safari (нет ffmpeg-генерации)
- **R5-Q6** — `sort=rating_user` возвращает непробованные в конец без разделителя

## Запланировано (next)

### Бэк-фундамент без UI — нужно дотянуть UI

- [ ] Кнопка «по оценке N» в шапке места (бэк уже отдаёт `ratings_per_user`)
- [ ] Полная интеграция `attended_by` / `visit_from-to` в UI фильтров

### Инфраструктура

- [ ] CI/CD (GitHub Actions: линт + тесты + docker build)
- [ ] Продакшн-деплой на VPS (есть `docker-compose.prod.yml`, не прокачано)
- [ ] Бэкап БД по расписанию (cron + pg_dump → S3/local)
- [ ] pgAdmin в `docker-compose.yml` (веб-интерфейс БД для разработки)
- [ ] Rate limiting на API (защита от спама)
- [ ] ffmpeg-генерация постеров для видео-кружков (решает R5-Q5 на iOS)
- [ ] HTTPS-туннель для записи видео через WiFi (Tailscale)

### Админка

- [ ] Панель superuser: CRUD кухонь/категорий/тегов из UI
- [ ] Управление пользователями: блокировка, смена роли (API готов — `/api/admin/users`, нет UI)
- [ ] Модерация отзывов: просмотр всех, удаление неуместных
- [ ] Дашборд: кол-во заведений/отзывов/юзеров, графики

### Заведения

- [ ] Статус заведения: открыто / временно закрыто / закрылось навсегда
- [ ] Часы работы (расписание по дням)
- [ ] Средний чек (диапазон цен)
- [ ] Ссылка на меню / фото меню
- [ ] Контакты (телефон, Instagram, Telegram)
- [ ] Пользовательские теги (например: «есть веранда», «подходит для свиданий», «шумно»)

### Отзывы

- [ ] Комментарии к отзывам (обсуждение с друзьями)
- [ ] Reactions (C5) — `жму руку` / `тоже хочу` / `там был` штампики (после MVP)

### UI/UX

- [ ] Тёмная тема (G2) — токены подготовлены, фактической темы нет (отложено в next-next, ~3 мес после запуска)
- [ ] Page-transitions между маршрутами
- [ ] Свайпы (мобильные)
- [ ] «Ваш год / сезон» (C4) — раз в полгода открытка-итог (через ~3 мес после запуска)

### Качество

- [ ] Фронтенд-тесты (vitest / Playwright) — сейчас 0 тестов
- [ ] 5 eslint-warnings (`LocationPicker.vue`, `PlaceDetail.vue`)
- [ ] `components/LocationPicker.vue` — единственный неперевёрстанный Bootstrap-компонент
