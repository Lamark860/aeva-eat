# Правила использования Яндекс Карт в AEVA Eat

> Источник: https://yandex.ru/legal/maps_api  
> Дата ознакомления: 13 апреля 2026 г.

## Подключённые API

- **JavaScript API Яндекс.Карт** (включая Геокодер) — API-ключ в `frontend/index.html`

## Обязательные условия (бесплатная лицензия)

### 1. Открытый доступ
Сайт должен быть доступен любому пользователю. Регистрация допустима, но без особых ограничений (без платы, без паспортных данных). **Мы соответствуем** — обычная регистрация email+пароль.

### 2. Данные — только на карте Яндекса
Результаты геокодера и маршрутизации обязаны отображаться на карте Яндекса (JS API). Нельзя показывать их на картах других провайдеров (Google, Leaflet/OSM и т.д.). **Мы соответствуем** — используем только Яндекс Карты.

### 3. Запрет сохранения данных API
Данные, полученные через API (геокодер, маршруты), нельзя сохранять в свою БД. Кэширование ответов разрешено до **30 дней**.

> **Наш подход**: при создании заведения пользователь кликает по карте — координаты берутся из события `click` на карте, а не из ответа геокодера. Геокодер используется только для поиска позиции на карте и обратного определения адреса для отображения.

### 4. Копирайты и логотип
Запрещено скрывать, перекрывать или изменять логотип Яндекса и копирайты на карте. **Не трогаем стандартные элементы.**

### 5. Не для мониторинга/диспетчеризации
Нельзя отслеживать транспорт или сотрудников в реальном времени. **Не наш случай.**

### 6. Лимиты (сутки)
| Сервис | Лимит |
|--------|-------|
| JavaScript API (всё) | 25 000 запросов |
| HTTP Геокодер (входит в 25k) | 1 000 запросов |

Для личного проекта — более чем достаточно.

### 7. Ссылка на Условия
Нужно разместить ссылку на https://yandex.ru/legal/maps_termsofuse в доступном месте.
**Добавлена в footer приложения.**

## Конфиденциальность

Яндекс собирает IP-адрес и геолокацию посетителей нашего сайта через JS API. Мы:
- Не передаём Яндексу никаких персональных данных пользователей (email, имена) за пределами того, что API собирает автоматически
- Не используем Яндекс.Метрику или другие инструменты аналитики Яндекса сверх карт
- Информируем пользователей о использовании Яндекс Карт (ссылка в footer)

## Что нам ЗАПРЕЩЕНО

- ❌ Сохранять ответы геокодера в БД (координаты из геокодера)
- ❌ Показывать данные Яндекса на других картах
- ❌ Скрывать логотип/копирайт Яндекса
- ❌ Модифицировать код или данные API
- ❌ Перепродавать решение с Яндекс Картами
- ❌ Превышать лимиты запросов

## Что нам РАЗРЕШЕНО

- ✅ Использовать карту для показа заведений
- ✅ Геокодер для поиска адресов (с отображением на карте Яндекса)
- ✅ Координаты из клика по карте сохранять в свою БД
- ✅ Кэшировать ответы геокодера до 30 дней
- ✅ Кастомные маркеры поверх карты
